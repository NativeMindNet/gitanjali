# Команда: /translate.sanscrit

## Назначение

Универсальная команда для многоязычного перевода священных текстов с использованием нескольких источников истины одновременно. Перевод выполняется с глубиной понимания, эквивалентной тому, как если бы человек одновременно прочитал текст на всех современных языках и описал его смысл на языке назначения.

## Синтаксис

```
/translate.sanscrit <input> <output> --languages=<lang1,lang2,...>
```

### Параметры

| Параметр | Обязательный | Описание |
|----------|--------------|----------|
| `<input>` | Да | Путь к файлу или папке с исходными данными |
| `<output>` | Да | Путь к файлу или папке для результата |
| `--languages` | Да | Список целевых языков через запятую |

### Примеры

```bash
# Один файл → один файл
/translate.sanscrit original.json translation.json --languages=ko,th,zh-CN,zh-TW,ja

# Папка → папка
/translate.sanscrit original/ translation/ --languages=ko,th,zh-CN,zh-TW

# Только отдельные языки
/translate.sanscrit source.json result.json --languages=ja,vi
```

## Архитектура перевода

### Принцип множественных источников истины

Все исходные языки загружаются **одновременно** и используются как **параллельные источники смысла**:

```
┌─────────────────────────────────────────────────────────────┐
│                    ВХОДНЫЕ ДАННЫЕ                          │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐       │
│  │ Русский │  │ English │  │ Deutsch │  │ Español │       │
│  │  (ru)   │  │  (en)   │  │  (de)   │  │  (es)   │       │
│  └────┬────┘  └────┬────┘  └────┬────┘  └────┬────┘       │
│       │            │            │            │             │
│       └────────────┴────────────┴────────────┘             │
│                         │                                  │
│                         ▼                                  │
│              ┌───────────────────┐                        │
│              │ СИНТЕЗ ПОНИМАНИЯ  │                        │
│              │ (все смыслы       │                        │
│              │  одновременно)    │                        │
│              └─────────┬─────────┘                        │
│                        │                                   │
│                        ▼                                   │
│              ┌───────────────────┐                        │
│              │    Санскрит       │◄── ПОСЛЕДНИЙ           │
│              │  (только формы,   │    (термины, имена,    │
│              │   термины)        │     правильные формы)  │
│              └─────────┬─────────┘                        │
│                        │                                   │
│                        ▼                                   │
│              ┌───────────────────┐                        │
│              │  ГЕНЕРАЦИЯ        │                        │
│              │  ПЕРЕВОДА         │                        │
│              └───────────────────┘                        │
└─────────────────────────────────────────────────────────────┘
```

### Приоритет источников

1. **Современные языки (одновременно)**:
   - Русский (ru) — глубина философского понимания
   - Английский (en) — точность терминологии
   - Немецкий (de) — структурная ясность
   - Испанский (es) — эмоциональные оттенки

2. **Санскрит (последний)**:
   - НЕ используется для смысла
   - Используется ТОЛЬКО для:
     - Правильного написания имён собственных
     - Точной транслитерации терминов
     - Проверки форм составных слов

## Исполнение в отдельном контексте

### Изолированный агент

Перевод выполняется в **отдельном агенте** с чистым контекстом:

```
┌────────────────────────────────────────┐
│  ГЛАВНЫЙ АГЕНТ                         │
│  - Парсит аргументы                    │
│  - Загружает все исходные тексты       │
│  - Готовит пакет данных                │
│  - Запускает агент перевода            │
│  - Сохраняет результат                 │
└───────────────┬────────────────────────┘
                │
                ▼
┌────────────────────────────────────────┐
│  АГЕНТ ПЕРЕВОДА (отдельный контекст)   │
│  - Получает ВСЕ данные разом           │
│  - Синтезирует понимание               │
│  - Генерирует переводы                 │
│  - Возвращает результат                │
└────────────────────────────────────────┘
```

### Формат передачи данных агенту

```json
{
  "source_texts": {
    "ru": "Текст на русском...",
    "en": "English text...",
    "de": "Deutscher Text...",
    "es": "Texto en español..."
  },
  "sanskrit": {
    "text": "संस्कृत पाठ...",
    "terms": ["термин1", "термин2"],
    "names": ["имя1", "имя2"]
  },
  "target_languages": ["ko", "th", "zh-CN", "zh-TW", "ja"],
  "context": {
    "type": "sloka|title|vocabulary",
    "chapter": 7,
    "number": "7.1"
  }
}
```

## Качество перевода

### Принцип человеческого понимания

Перевод должен быть таким, как если бы:

> **Образованный человек прочитал текст на 4+ языках одновременно,
> синтезировал глубокое понимание смысла,
> и затем описал этот смысл на языке назначения
> с учётом культурного контекста целевой аудитории.**

### Критерии качества

| Аспект | Требование |
|--------|------------|
| Смысл | Полная передача философской глубины |
| Термины | Сохранение санскритских терминов с объяснением |
| Имена | Транслитерация + локальная традиция написания |
| Стиль | Соответствие священным текстам на языке назначения |
| Культура | Учёт религиозной традиции целевой культуры |

## Формат вывода

### Структура результата

```json
{
  "meta": {
    "source": "<input_path>",
    "generated": "<timestamp>",
    "languages": ["ko", "th", "zh-CN", "zh-TW"],
    "approved": false
  },
  "translations": {
    "<item_id>": {
      "ko": { "text": "...", "approved": false },
      "th": { "text": "...", "approved": false },
      "zh-CN": { "text": "...", "approved": false },
      "zh-TW": { "text": "...", "approved": false }
    }
  }
}
```

### Типы выходных данных

#### Для файла главы (chapter-XX.json)

```json
{
  "chapter": 7,
  "title": {
    "ko": "지식과 깨달음의 요가",
    "th": "โยคะแห่งความรู้และการตระหนักรู้"
  },
  "slokas": {
    "7.1": {
      "ko": "지고하신 주께서 말씀하셨다...",
      "th": "พระผู้เป็นเจ้าตรัสว่า..."
    }
  }
}
```

#### Для словаря (vocabulary)

```json
{
  "vocabulary": {
    "word_id": {
      "meaning": {
        "ko": "의미...",
        "th": "ความหมาย..."
      },
      "transliteration": {
        "ko": "транслитерация на хангыль",
        "th": "транслитерация на тайский"
      }
    }
  }
}
```

## Поддерживаемые языки

### Источники (input)

| Код | Язык | Приоритет |
|-----|------|-----------|
| ru | Русский | 1 (основной) |
| en | English | 2 |
| de | Deutsch | 3 |
| es | Español | 3 |
| sa | संस्कृत | Последний (только формы) |

| Код | Язык | Особенности |

| ko | 한국어 | Буддийская терминология |
| th | ภาษาไทย | Тхеравадинская традиция |
| zh-CN | 简体中文 | Даосские параллели |
| zh-TW | 繁體中文 | Традиционные формы |
| ja | 日本語 | Дзен-буддийский контекст |
| vi | Tiếng Việt | Махаянская традиция |
| hi | हिन्दी | Родная традиция |
| bn | বাংলা | Бенгальский вайшнавизм |

## Валидация

### Автоматическая проверка

После генерации перевода:
- ✓ JSON валиден
- ✓ Все целевые языки присутствуют
- ✓ Нет пустых переводов
- ✓ Санскритские термины сохранены
- ✓ Имена собственные транслитерированы

## Ограничения

- **Encoding**: UTF-8
- **Размер файла**: < 200000 символов (если больше, необходимо дробить на чанки)
- **Параллельность**: 1 агент на задачу
