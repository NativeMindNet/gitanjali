# Visual Mockups: Gitanjali Adaptive Layout

> Version: 1.0
> Status: DRAFT
> Last Updated: 2026-05-01

## Overview

ASCII макеты адаптивной верстки для Sri Gaudiya Gitanjali. Показаны breakpoints для разных размеров экранов и все состояния UI.

---

## Breakpoints

| Категория | Ширина | Описание |
|-----------|--------|----------|
| XS | < 360px | Старые/маленькие телефоны |
| SM | 360-599px | Стандартные телефоны |
| MD | 600-899px | Планшеты, большие телефоны в landscape |
| LG | >= 900px | Десктоп, большие планшеты |

---

## Screen: Reader Page

### XS Layout (< 360px) - Компактный

```
+--------------------------------+
| = Sri Gaudiya Gitanjali    [S] |  <- [S] = Settings icon
+--------------------------------+
| Стр. 42                        |
+--------------------------------+
|                                |
| +----------------------------+ |
| |                            | |
| |    [Background Image]      | |
| |                            | |
| +----------------------------+ |
|                                |
| ~~~~~~~~~~~~~~~~~~~~~~~~~~~~   |  <- Text content (scrollable)
| ~ Песня номер 12             ~ |
| ~ Автор: Бхактивинода Тхакур ~ |
| ~                            ~ |
| ~ Текст песни идет здесь     ~ |
| ~ с переносами для узкого    ~ |
| ~ экрана...                  ~ |
| ~~~~~~~~~~~~~~~~~~~~~~~~~~~~   |
|                                |
| +----------------------------+ |
| | Комментарий:               | |  <- Comments card (collapsible)
| | Эта песня написана...      | |
| +----------------------------+ |
|                                |
+--------------------------------+
| [<] [>] [H] [Q] [B] [A]        |  <- Compact toolbar
+--------------------------------+
```

**Легенда XS toolbar:**
- `[<]` Previous page
- `[>]` Next page
- `[H]` Home/TOC
- `[Q]` Search
- `[B]` Bookmarks
- `[A]` Audio (if available)

## Component: Page Navigation Links

Некоторые страницы содержат ссылки на другие страницы (TOC, etc).

```
+------------------------------------------+
|  Навигация:                              |
|                                          |
|  +--------+  +--------+  +--------+      |
|  | [img]  |  | [img]  |  | [img]  |      |
|  | Intro  |  | Ch. 1  |  | Ch. 2  |      |
|  +--------+  +--------+  +--------+      |
|                                          |
+------------------------------------------+
```

На маленьких экранах - горизонтальный scroll:

```
+---------------------------+
|  Навигация:               |
|  +------+ +------+ +------+---> (scroll)
|  |[img] | |[img] | |[img] |
|  |Intro | |Ch. 1 | |Ch. 2 |
|  +------+ +------+ +------+
+---------------------------+
```

## Flow: Main Navigation

```
[Reader Page] --(swipe left)---> [Next Page]
      |
      +--(swipe right)---> [Previous Page]
      |
      +--(tap [S])---> [Settings Modal]
      |                      |
      |                      +--(select language)---> [Reload Content]
      |                      |
      |                      +--(close)---> [Reader Page]
      |
      +--(tap [Search])---> [Search Sheet]
      |                          |
      |                          +--(tap result)---> [Reader Page @ result]
      |
      +--(tap [Bookmarks])---> [Bookmarks Sheet]
      |                             |
      |                             +--(tap bookmark)---> [Reader Page @ bookmark]
      |
      +--(tap [Home])---> [Reader Page @ TOC]
      |
      +--(tap [Audio])---> [Toggle Audio Playback]
      |
      +--(tap Page Link)---> [Reader Page @ linked page]
```

## Responsive Behavior Summary

| Element | XS (<360) | SM (360-599) | MD (600-899) | LG (>=900) |
|---------|-----------|--------------|--------------|------------|
| Header | Compact, icon only | Full title | Full title | Full title |
| Content padding | 8px | 16px | 24px | 32px |
| Max content width | 100% | 100% | 100% | 800px |
| Toolbar buttons | Icons only | Icons + short text | Icons + text | Full text |
| Font scale | 0.9x | 1.0x | 1.0x | 1.1x |
| Page links | Horizontal scroll | Horizontal scroll | Grid | Grid |

---

## Notes

- Все модальные окна (settings, search, bookmarks) используют bottom sheet для единообразия
- Audio player встроен в toolbar area, чтобы не занимать место контента
- На XS экранах приоритет читаемости текста над декоративными элементами
- Swipe-навигация работает на всех размерах экрана
- Safe area respected на всех платформах (notch, navigation bar)
